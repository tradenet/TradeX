Manage Saved Links
Table of Contents
Overview
The Manage Saved Links interface allows you to view, edit, delete, and create toplists from your saved links. Saved links are promotional links created with the Link Generator that have custom thumbnail images and can be combined into toplists. Unlike trade toplists, saved link toplists are completely separate from your trades and are used for promotional content that goes through your tracking system (out.php) to content URLs, skim schemes, or specific trades.
Saved Links List
The main view shows all your saved links in a table with the following columns:
    Link ID - The unique identifier for this link (used in the Link Generator's "Link Identifier" field).
    Link Name - The display name for this link, which appears in the toplist and management interface.
    Type - The type of link: Percent (% to content), Scheme (skim scheme), or Trade (specific trade).
    Thumbnails - The number of custom thumbnail URLs saved for this link.
    Functions - Available actions for each saved link:
    • Edit - Modify the link name and custom thumbnail URLs
    • Delete - Remove the saved link (requires confirmation)
    • Copy Toplist URL - Get a toplist URL for just this single link
Editing Saved Links
You can edit the following fields for a saved link:
    Link Name - Change the display name that appears in the toplist.
    Custom Thumbnails - Add, remove, or modify the thumbnail URLs (one per line). Each URL should be a complete URL starting with http:// or https://. These images will be displayed in the toplist and will link through out.php to the link's destination.

Note: The Link ID, Type, and link settings (percent, skim scheme, content URL, etc.) cannot be modified after creation. If you need to change these, create a new saved link with the Link Generator.
Generate Toplist HTML
At the bottom of the Manage Saved Links page, you can create a toplist from multiple saved links:
  1. Select one or more saved links from the dropdown list (hold Ctrl/Cmd to select multiple)
  2. Click "Generate Toplist URL"
  3. Copy the generated URL and use it to display your toplist

The generated toplist URL format is:
http://yourdomain.com/savedlinks.php?links=link1,link2,link3

When visitors access this URL, they'll see a toplist displaying the custom thumbnails for each saved link. Clicking a thumbnail will:
  1. Go through out.php (tracking the click with the link's identifier)
  2. Redirect to the link's destination (content URL, skim to trades, or specific trade)

This allows you to track clicks on promotional content separately from your trade traffic while still using your tracking system and skim schemes.
Creating Saved Links
To create a new saved link, use the Link Generator from the Tools menu:
  1. Configure your link settings (type, content URL, skim scheme, etc.)
  2. Provide a Link Identifier (required for saved links)
  3. Check the box "Save this link with custom thumbnails"
  4. Enter a Link Name (how it will appear in the toplist)
  5. Add Custom Thumbnail URLs (one per line)
  6. Click "Generate"

The link will be saved and will appear in the Manage Saved Links interface where you can later edit the thumbnails, combine it with other saved links in a toplist, or delete it.
Use Cases
Saved links with toplists are useful for:
  • Creating promotional galleries with custom graphics that link to sponsor content
  • Building themed toplists with seasonal or event-specific imagery
  • Displaying "featured content" with custom thumbnails separate from your trades
  • Creating multiple toplist variations for different pages on your site
  • Tracking which promotional graphics get the most clicks
  • Running A/B tests with different thumbnail images for the same content

Since all clicks go through out.php with the link identifier, you can see detailed tracking stats for each saved link in your stats pages.
