Custom Thumbnail Toplist Generator
Overview

The Custom Thumbnail Toplist Generator allows you to create toplists with custom thumbnail images for each trade. This is useful when you want to display specific promotional images instead of the standard thumbnail rotation.

How to Use
  1. Access the Tool: Go to Tools > Custom Thumb Toplist Generator
  2. Filter Trades (Optional): Use the Group and Category filters to narrow down the trade list
  3. Select Trades: Choose one or more trades from the list and click "Add Selected Trades â†’"
  4. Add Custom Thumbnails: For each trade, enter custom thumbnail URLs (one per line):
    • URLs must be complete (start with http:// or https://)
    • Each line represents one thumbnail image
    • You can add multiple thumbnails per trade
  5. Save: Click "Save Custom Thumbnails" to save the URLs to your trades
  6. Create Toplist: Go to Toplists and create a new toplist:
    • Set Source to "Template"
    • Use the template: toplist-random-36-custom-thumbs.tpl
    • Or create your own template based on this one
  7. Build & Deploy: Build the toplist and it will use your custom thumbnails
Example Custom Thumbnails

For a trade, you might enter: 

https://example.com/promo/banner1.jpg
https://example.com/promo/banner2.jpg
https://example.com/promo/banner3.jpg

The toplist will randomly display one of these three images for that trade.

How It Works

When you save custom thumbnails:

  • The URLs are stored in the trade's custom_thumbs field
  • The toplist template retrieves these URLs when building
  • Each trade displays one of its custom thumbnails randomly
  • If a trade has no custom thumbnails, it shows a default "No Thumb" image
Tips
  • Image Dimensions: Keep all thumbnails the same size for a consistent appearance
  • File Hosting: Ensure thumbnail images are hosted on a reliable server
  • HTTPS: Use HTTPS URLs to avoid mixed content warnings
  • Testing: Build a test toplist before deploying to your live site
  • Updating: You can edit custom thumbnails anytime and rebuild the toplist
Custom Template

The included template (toplist-random-36-custom-thumbs.tpl) displays up to 36 thumbnails in a 6-column grid. You can create custom templates based on this one to:

  • Change the layout (different grid sizes, carousel, etc.)
  • Add additional styling or effects
  • Include additional trade information
  • Adjust the fallback behavior when no custom thumbs exist

See the Toplist Management documentation for more information about creating custom toplist templates.
